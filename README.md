# Habit Quest

A private points-based habit tracker that stores everything on the device.

## Fastest way to run it

1. Upload all files in this folder to a static web host such as GitHub Pages, Netlify, or Cloudflare Pages.
2. Open the resulting URL in Safari on iPad.
3. Tap Share → Add to Home Screen.
4. Launch it like a normal app.

## Included in version 1

- Good habits earn points
- Bad habits subtract points
- Editable point values
- Daily checkoffs
- 7-day score chart
- Positive-day streak
- Offline support after first load
- No account and no server
