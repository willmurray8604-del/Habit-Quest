# Habit Quest v3.1

New in this release:
- Daily notes that save and sync automatically
- Per-day count/pause toggle for every good or bad habit
- Paused good habits are removed from that day's possible-point total
- Individual current streak beside every habit
- Individual best streak in Statistics
- Good-habit point values are green
- Grade ring changes color with the grade
- Calendar days use the actual grade color
- Existing Firebase sync, history, settings, and local data remain compatible

Upload all seven files to the root of the existing GitHub repository and replace the previous versions.
